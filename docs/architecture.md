# Architecture

## Overview

Hybrid DEX is a decentralized exchange platform combining on-chain settlement with off-chain matching.

## Components

- **Matching Engine**: Rust-based high-performance order matching
- **API Gateway**: Node.js Fastify service for REST and WebSocket APIs
- **Smart Contracts**: Solidity contracts for settlement and vault management
- **Web Trader**: React-based trading interface
- **Admin Console**: Admin dashboard for operations
